# Demo for hydra

## How to Set Up the Project
1. Clone this branch:
```bash
git clone https://github.com/khuyentran1401/hydra_demo.git
```
2. Install [Poetry](https://python-poetry.org/docs/#installation)
3. Set up the environment:
```bash
make setup
```
